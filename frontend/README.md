# Frontend (game client)

Vite + React + TypeScript client for **Hunter vs Runner**: **React Three Fiber** for the 3D arena, **Colyseus** for realtime play, **Supabase** for auth.

**Environment variables, dev server, and build** are documented in the [repository root README](../README.md#first-time-setup).

Entry points: `src/main.tsx`, `src/App.tsx`.
